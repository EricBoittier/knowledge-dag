# GUIs

This domain groups the browser and local UI surfaces in the repository.

## Current Surfaces

- `knowledge-web/`: the original knowledge DAG web application at the repository root.
- `ai-director/`: the local AI Director browser shell in `../ai-director-app/apps/desktop`.
- `dataset-tools/`: voice dataset capture and SRT dataset generation UIs.

These folders document the current UI entrypoints. They do not move the runnable apps yet.
