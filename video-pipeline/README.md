# Video Pipeline (Linux + DaVinci MVP)

This project provides a low-risk, editor-first pipeline:
- normalize clips into DaVinci-friendly mezzanine media
- validate media streams before import
- generate a sequential FCPXML timeline

## Why this format

Default mezzanine policy:
- Container: `MOV`
- Video: `DNxHR HQ` (`dnxhd` + `dnxhr_hq`, `yuv422p`)
- Audio: `PCM s16le`, 48kHz stereo

This favors import reliability in DaVinci Resolve on Linux.

Optional review output:
- Container: `MP4`
- Video: `AV1` (`libsvtav1`)
- Audio: `PCM s16le`

## Repository layout

- `project_config.json`: pipeline config and media policy
- `src/media_probe.py`: ffprobe validation helpers
- `src/normalize_clips.py`: clip normalization + manifest writing
- `src/build_timeline_fcpxml.py`: sequential FCPXML generation
- `src/run_pipeline.py`: one-command orchestration
- `scripts/test_import_check.sh`: pre-import stream checks
- `templates/timeline_base.xml`: optional XML template notes
- `output/`: generated files

## Requirements

- Python 3.10+
- `ffmpeg` and `ffprobe` available in PATH

## Quick start

1. Put source videos in `./input` (or override via `--input-dir`).
2. Run:

```bash
python3 src/run_pipeline.py --config ./project_config.json
```

For a more visually edited demo timeline (reordered + trimmed clips):

```bash
python3 src/run_pipeline.py --config ./project_config.json --showcase
```

To remove stale normalized/review files before regenerating:

```bash
python3 src/run_pipeline.py --config ./project_config.json --force --clean-old-normalized --showcase
```

Outputs:
- normalized clips in `output/normalized`
- manifest at `output/normalized/manifest.json`
- timeline at `output/timeline_davinci_resolve.fcpxml`
- showcase manifest (when enabled): `output/normalized/manifest.showcase.json`

## Explicit policy ffmpeg commands

### Normalize to DaVinci-safe MOV (DNxHR + PCM)

```bash
ffmpeg -y -i INPUT \
  -map 0:v:0 -map 0:a:0 \
  -c:v dnxhd -profile:v dnxhr_hq -pix_fmt yuv422p -r 30 \
  -c:a pcm_s16le -ar 48000 -ac 2 \
  OUTPUT.mov
```

### Optional review file (AV1 + PCM)

```bash
ffmpeg -y -i INPUT \
  -map 0:v:0 -map 0:a:0 \
  -c:v libsvtav1 -crf 30 -preset 6 \
  -c:a pcm_s16le \
  OUTPUT.review.mp4
```

## DaVinci import flow

1. Import normalized media first.
2. Import `output/timeline_davinci_resolve.fcpxml`.
3. If media relinking is needed, use absolute paths generated by this pipeline.
